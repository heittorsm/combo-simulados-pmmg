# Combo de Simulados — PMMG

Aplicativo web para acompanhamento de desempenho em simulados de concursos.

## Características

- **Sem cadastro**: dados salvos automaticamente no navegador do aluno
- **Autossuficiente**: funciona offline, sem servidor
- **3 Simulados**: acompanhamento completo da evolução
- **Análise automática**: diagnóstico gerado por fórmulas

## Abas

1. **Como usar** — instruções e legenda de cores
2. **Evolução** — acertos, erros, em branco por disciplina; dashboard com gráficos
3. **Tempo de Prova** — controle de velocidade, estratégia e revisão
4. **Diagnóstico** — análise automática do desempenho
5. **Configurações** — faixas de cor personalizáveis, backup de dados

## Para usar

Abra o arquivo `combo-simulados-pmmg.html` no navegador (desktop, tablet ou celular).

## Deploy local

Simples: copie o arquivo e compartilhe com os alunos por e-mail, WhatsApp ou área de membros.
